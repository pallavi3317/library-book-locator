# Library Book Locator

## Project Description

Library Book Locator is a web-based application developed using HTML, CSS and JavaScript.

The system helps users:

- Search books
- Check book availability
- View book categories
- View book details
- Admin book management

## Technologies Used

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Project Pages

- Home
- Search Books
- Categories
- Book Details
- Admin Login
- Admin Dashboard
